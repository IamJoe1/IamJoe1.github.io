3D Modeling

Using Blender.

Renders

Low Poly Environment

This is a low poly environment that I created in the Blender Cycles render engine. This render uses a variety of objects including trees, mountains, ponds and a Sun with an empahsis and the subtile lighting and reflections. It utizlizes an HDRI to give the enviornment an appropirate, well-rounded light source that adds a natural feel to the viewer. The sky is the cherry on top, which uses an imported image as a plane, and is positioned behind the model in order to act as a background.

Realistic Environment

With this render, I went for a more realistic apporch as oppsed to the previous one. This project was created using Blenders Eevee render engine and focuses on texture, bloom, and reflections to emphasize the sci-fi style.
